# FT-Logistika

Starter repository for the FT-Logistika platform.

## Structure
- `backend/` FastAPI + SQLAlchemy + Alembic
- `frontend/` React + Vite
- `infra/` service configs / CI / reverse proxy

## Quick start (backend)
1. Create `.env` from `.env.example`
2. Create virtualenv and install requirements
3. Run migrations with Alembic
4. Start API with Uvicorn
